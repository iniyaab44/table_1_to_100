# Tables 1 to 100: Interactive Multiplication Learning Tool

An interactive, high-performance web application designed to help children and learners master multiplication tables from 1 to 100. Featuring a bold Neo-Brutalist design, offline-ready speech, and a structured 2-column layout for optimal readability.

## 🚀 Installation Steps

### 1. Prerequisites
- **Node.js** (v18 or higher)
- **npm** or **yarn**

### 2. Clone and Install
```bash
# Clone the repository (if applicable)
# git clone <repository-url>
# cd <project-directory>

# Install dependencies
npm install
```

### 3. Run the Development Server
```bash
npm run dev
```
The application will be available at `http://localhost:3000`.

### 4. Build for Production
```bash
npm run build
```

## 🎯 Key Features
- **Zero API Dependency**: Uses the browser's native Web Speech API (works offline).
- **Tables 1-100**: Comprehensive coverage for early learners.
- **Pagination**: 10 tables per page for a focused learning experience.
- **Search**: Instantly find any table from 1 to 100.
- **Visual Clarity**: 4px black divider lines and bold typography.

## ❓ Why is this an NPM/React project?
This is a **Web Application**. While Python is great for backend scripts, modern interactive user interfaces (like this one with animations and real-time search) are best built using **React and TypeScript**. 
- `npm install` is used to manage the frontend libraries (icons, animations, layout).
- This allows the app to run directly in any browser (Chrome, Firefox, Safari) without needing a Python backend.
